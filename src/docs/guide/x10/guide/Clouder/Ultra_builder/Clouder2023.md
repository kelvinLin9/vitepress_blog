## 3. 企業開出懸賞案件、任何人都可以申報取得報酬

- YP或其他企業方可以根據自己的需求和項目開出懸賞案件，並設定報酬金額。

- 任何人都可以申報參與項目，並提交相關方案和實現方案。

> 可以通過參與懸賞項目獲得報酬，展示自己技能經驗，吸引更多客戶機會。

> 使用 sats、接通nostr、智能合約
